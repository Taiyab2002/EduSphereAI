## Project Overview

EduSphere AI is a modern cloud-based Learning Management System (LMS) that enables students and instructors to interact through a secure and scalable web platform.

The system provides authentication, role-based authorization, course management, personalized dashboards, and cloud database integration using modern Full Stack technologies.

---

## Key Features

- Secure User Authentication using JWT
- Role-Based Access Control (RBAC)
- Student, Teacher and Admin Roles
- Course Creation & Management
- My Courses Dashboard
- Responsive User Interface
- RESTful API Architecture
- MongoDB Atlas Cloud Database
- Production Deployment (Vercel + Render)

## Installation

### Clone Repository

```bash
git clone https://github.com/Taiyab2002/EduSphereAI.git
```

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
